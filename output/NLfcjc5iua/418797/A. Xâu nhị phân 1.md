# A. Xâu nhị phân 1

## 📖 Problem

Cho số nguyên
$n$
, hãy xuất ra các xâu nhị phân có độ dài
$n$
theo thứ tự từ điển.


## 🧩 Input

Input
Một dòng duy nhất chứa số nguyên
$n$
$(1 ≤n≤ 18)$


## 💡 Output

Output
Gồm nhiều dòng, mỗi dòng là xâu nhị phân.


## 🧠 Example

### Input

```text
3
```

### Output

```text
000
001
010
011
100
101
110
111
```
