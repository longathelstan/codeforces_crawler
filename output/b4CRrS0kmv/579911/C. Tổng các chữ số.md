# C. Tổng các chữ số

## 📖 Problem

Gọi
$f(i)$
là tổng các chữ số của
$i$
. Cho
$2$
số
$L$
và
$R$
Yêu cầu:
Hãy tính
$f(L) +f(L+ 1) + ... +f(R)$


## 🧩 Input

Input
Dòng đầu gồm
$2$
số
$L$
và
$R$
$(1 ≤L≤R≤ 10100)$


## 💡 Output

Output
Gồm một dòng duy nhất là đáp án cần tìm chia dư cho
$109+ 7$


## 🧠 Example

### Input

```text
6 20
```

### Output

```text
87
```
