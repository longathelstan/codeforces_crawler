# B. SumLR (7 điểm)

## 📖 Problem

Cho dãy số
$a1,a2, …,an$
và
$2$
số nguyên dương
$l,r$
. Tìm dãy con liên tiếp thỏa mãn:
*
Số lượng phần tử nằm trong đoạn
$[l,r]$
*
Tổng các phần tử đạt giá trị lớn nhất.


## 🧩 Input

Input
Dòng đầu tiên ghi
$3$
số nguyên dương
$n,l,r$
$(1 ≤l≤r≤n≤ 3 * 106)$
Dòng thứ 2 chứa
$n$
số nguyên xác định dãy số
$a1,a2, …,an$
$( - 109≤ai≤ 109)$


## 💡 Output

Output
Gồm một dòn duy nhất là đáp án cần tìm


## 🧠 Example

### Input

```text
9 2 3
40 -40 0 3 -5 0 3 0 1
```

### Output

```text
4
```
